# Advanced Biomass - Astro Site

Modern static site conversion of advancedbiomass.com built with:
- Astro + Tailwind CSS
- Decap CMS for content management
- Deployed to v2.advancedbiomass.com

## Quick Start
```bash
npm install
npm run dev
```

## Deploy
```bash  
npm run build
# Upload dist/ folder to hosting
```

CMS available at: /admin
